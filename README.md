# Uzcarsale — Chevrolet в Узбекистане

Статический сайт-каталог объявлений о продаже автомобилей Chevrolet в Узбекистане.
Дизайн построен по дизайн-системе в стиле Lamborghini.com (см. `docs/DESIGN.md`):
чёрная кинематографичная сцена, один шрифт Barlow Condensed 400, UPPERCASE-заголовки,
нулевые радиусы, без теней и один жёлтый акцент `#ffc000` на экран.

## Страницы

- `index.html` — главная: hero, статистика, модельный ряд, свежие объявления
- `catalog.html` — каталог с фильтрами (модель, состояние, регион, КПП, цена, сортировка)
- `car.html?id=N` — карточка объявления: характеристики, продавец, похожие объявления

## Структура

```
css/styles.css   — дизайн-система (токены + компоненты)
js/data.js       — данные: 10 моделей Chevrolet, 22 объявления
js/app.js        — рендер и фильтрация (vanilla JS, без зависимостей)
docs/DESIGN.md   — дизайн-референс
SITE_CONTRACT.md — контракт команды агентов (классы, схема данных)
```

## Запуск

Сборка не нужна — любой статический сервер:

```
python -m http.server 8000
```

Сайт сделан командой субагентов Claude Code: design-system, data, home, catalog.
